#### How to run
+ **Preliminary**
    + install cmake >= 3.15
    + C++ Compiler supports at least C++14
+ **Build and Compile**
    ```shell
    cd build
    cmake ..
    cmake --build . --parallel 8
    ```
+ **Run**
    ```shell
    cd bin/Debug
    ./project1.exe
    ```
